# Seekcy Beacon SDK for iOS#

## Overview ##

Seekcy Beacon SDK for iOS是一个二次开发使用的库，以便与iBeacons进行互动。
该SDK系统要求iOS7及以上，设备要求iphone4S及以上。

[Seekcy SDK for Android](https://github.com/Seekcy/SeekcyBeacon_SDK_Android)这是Android SDK链接地址。

SDK包括的功能:

- Beacon 广播扫描
- Beacon 区域监听
- Beacon 连接、读取、配置
